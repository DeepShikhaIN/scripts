# Audacity

================

## Install

Install Audacity via an unofficial PPA.

### Step 0

```
sudo add-apt-repository ppa:ubuntuhandbook1/audacity -y
```

### Step 1

```
sudo apt update && sudo apt install audacity -y
```

Done!

## Update

```
sudo apt update && sudo apt upgrade -y
```

## Uninstall

```
sudo apt purge audacity -y
```

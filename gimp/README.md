# GIMP

=======

## Install

Install GIMP via an unofficial PPA.

### Step 0

```
sudo add-apt-repository ppa:ubuntuhandbook1/gimp -y
```

### Step 1

```
sudo apt update && sudo apt install gimp -y
```

Done!

## Update

```
sudo apt update && sudo apt upgrade -y
```

## Uninstall

```
sudo apt purge gimp -y
```
